# AI-Movie-Recommendation-System


A **Machine Learning-based Movie Recommendation System** that suggests movies to users based on **content similarity and genre preferences**. The system uses **Natural Language Processing (NLP)** and **Cosine Similarity** to analyze movie metadata such as genres, keywords, cast, and overview.

This project demonstrates the implementation of a **content-based recommendation engine** using Python and machine learning libraries.

# 🚀 Features

- Content-based movie recommendation
- Genre-based movie recommendation
- Movie rating display
- Pagination for multiple recommendations
- Interactive command line interface
- Loop-based recommendation system
- Uses real-world TMDB movie dataset

# 🧠 Machine Learning Concepts Used

- Natural Language Processing (NLP)
- Feature Engineering
- Bag of Words Model
- Count Vectorization
- Cosine Similarity
- Content-Based Filtering

# 📂 Dataset

This project uses the **TMDB 5000 Movie Dataset**.

Dataset files used:

- `tmdb_5000_movies.csv`
- `tmdb_5000_credits.csv`

Dataset Source:
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata
